\outl0\strokewidth0 \strokec2 Document Retrieval Backend\
\pard\pardeftab720\sa240\partightenfactor0

\f1\b0\fs24 \cf0 This project is a document retrieval backend that utilizes embeddings and vector search to retrieve relevant documents based on semantic similarity. It is built using 
\f0\b FastAPI
\f1\b0  and uses 
\f0\b FAISS
\f1\b0  for fast similarity search and 
\f0\b Redis
\f1\b0  for caching search results and tracking API usage.\
